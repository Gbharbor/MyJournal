Comandos Básicos do Git

Publicado em: 30/09/2024

Este post apresenta um guia rápido dos comandos mais essenciais do Git para quem está começando.

Comandos Essenciais

git init

Inicializa um novo repositório Git no diretório atual.

git clone [URL]

Clona um repositório existente a partir de uma URL.

git add [arquivo]

Adiciona arquivos ao estágio para commit.

git commit -m "[mensagem]"

Cria um commit com uma mensagem descritiva.

git status

Verifica o status dos arquivos no repositório.

git log

Exibe o histórico de commits do repositório.

git pull

Baixa e aplica as alterações do repositório remoto no local.

git push

Envia as alterações do repositório local para o remoto.

git branch

Lista as branches existentes no repositório.

git checkout [branch]

Troca para uma branch específica.

git merge [branch]

Faz merge de uma branch específica na branch ativa.

git remote add origin [URL]

Define o repositório remoto para onde o código será enviado.
